- ticket 上は `services/timecard.ts` + `services/aggregation.ts` の 2 ファイル分割が示されていたが、ユーザー指示および実装規模から `services/punches.ts` + `lib/time.ts` の 2 ファイルに統合した。集計ロジック (純関数) と DB 操作 (副作用) の分離は維持されている

- 深夜帯計算は走査範囲を `Math.ceil(duration/1day)+2` 日でループする実装。最大数日のシフトであっても定数オーダーで完了
