# task-4002 done_when

- `GET /api/auth/employees` がシードの従業員一覧を返す (退職者除外)
- 正しい PIN で `POST /api/auth/pin-login` が 200 を返し Set-Cookie に `hanare_sid` がある
- 誤った PIN で 401 + `remaining` フィールドが返る
- 同一従業員に 5 回連続で誤 PIN を送ると 6 回目以降 423 + `lock_until` が返る
- 5 分経過 (or DB の lock_until を過去に書き換え) で再度ログイン可能
- `POST /api/auth/admin-login` が `oyakata`/`hanare2026` で成功する
- ログイン後 `GET /api/auth/me` が現在ユーザーを返す
- `POST /api/auth/logout` 後は `/api/auth/me` が 401
- vitest で auth サービスの unit テストが pass
