# 打刻 API と集計ロジック

## 背景

打刻操作と当月集計を実装する。日跨ぎ勤務・深夜・残業計算を含む。

## 追加するファイル

- `src/server/routes/punches.ts`
- `src/server/services/timecard.ts`
- `src/server/services/aggregation.ts`
- `src/server/lib/time.ts` (純関数: minutesBetween, splitNightMinutes, computeOvertime 等)
- `tests/unit/time.test.ts`

## エンドポイント

- `POST /api/punches`: `{punch_type, store_id}` → 状態遷移検証 + 挿入 + メッセージ生成
- `GET  /api/punches/me?from&to`: 自分の打刻履歴
- `GET  /api/punches?store_id&employee_id&from&to`: manager+ 全打刻
- `GET  /api/punches/me/summary`: 当月集計 `{worked, overtime, break, night}`
- `GET  /api/punches/me/state`: 現在状態 `off|working|on_break` + 直近打刻

## 状態遷移ルール

- off → clock_in のみ可
- working → clock_out, break_start のみ可
- on_break → break_end のみ可
- 違反は 409 `invalid_transition`

## 集計ロジック (lib/time.ts)

- 1 勤務 = clock_in 〜 次の clock_out (日跨ぎ可、24h 超は警告対象)
- worked_minutes = (out - in) - 休憩合計
- overtime_minutes = max(0, worked - 480)
- night_minutes = 22:00–05:00 重複時間の合計 (両日に跨る場合は分割)
- すべて分単位整数で扱う
