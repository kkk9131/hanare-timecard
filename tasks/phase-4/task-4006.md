# エクスポート API

## 背景

`agent-docs/excel-export.md` の仕様で xlsx/CSV を生成し配信する。

## 追加するファイル

- `src/server/routes/exports.ts`
- `src/server/services/export.ts`
- `tests/unit/export.test.ts`

## エンドポイント

- `GET /api/exports/xlsx?from&to&store_id` (admin)
- `GET /api/exports/csv?from&to&store_id` (admin)

## 実装方針

- 期間内の `time_punches` を集計 → `WorkDayRow[]` を生成
- xlsx: exceljs で 1 シート、列定義・凍結ヘッダ・スタイル適用 (excel-export.md 表参照)
- csv: 自前で BOM + CRLF + ダブルクォートエスケープ
- レスポンスヘッダ:
  - xlsx: `Content-Type: application/vnd.openxmlformats-officedocument.spreadsheetml.sheet`
  - csv: `Content-Type: text/csv; charset=utf-8`
  - `Content-Disposition: attachment; filename="hanare-{store}-{YYYY-MM}.{ext}"`
- 全店舗指定 (`store_id` 省略) は `all` ファイル名
