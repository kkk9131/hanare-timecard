# task-4001 done_when

- `npm run dev` でサーバが起動し `0.0.0.0:3000` で listen する
- 起動時にコンソールに LAN IP URL (`http://192.168.x.x:3000` 形式) が表示される
- `curl http://localhost:3000/api/system/health` が `{"status":"ok","time":<number>}` を返す
- 不正な Cookie で認証必須エンドポイントを叩くと 401 が返る (テスト用ダミーエンドポイントで確認可)
- 起動時に DB マイグレーションが自動適用される
- 統一エラーフォーマット `{error,message}` で 4xx/5xx が返る
