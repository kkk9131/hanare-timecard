# プロジェクト初期化

## 背景

空ディレクトリの状態から、Hono + React + Vite + TypeScript + Drizzle + better-sqlite3 のモノレポ構成を立ち上げる。`agent-docs/architecture.md` の「ディレクトリ構成」セクションに従う。

## 追加するファイル

- `package.json` (scripts: dev, build, start, migrate, seed, lint, test)
- `tsconfig.json` + `tsconfig.server.json` + `tsconfig.client.json`
- `vite.config.ts` (root: `src/client`, build outDir: `../../dist/client`, dev proxy `/api → :3000`)
- `biome.json`
- `drizzle.config.ts`
- `.gitignore` (`node_modules`, `dist`, `data/*.db`, `data/backups`)
- `src/server/index.ts` (最小: `console.log('hanare booting')` のみで OK)
- `src/client/main.tsx` + `src/client/index.html` (最小)
- `src/shared/.gitkeep`

## 依存ライブラリ

- runtime: hono, @hono/node-server, better-sqlite3, drizzle-orm, zod, bcrypt, exceljs, react, react-dom, react-router-dom, @tanstack/react-query, zustand
- dev: typescript, vite, @vitejs/plugin-react, drizzle-kit, vitest, @biomejs/biome, @types/node, @types/react, @types/react-dom, @types/better-sqlite3, @types/bcrypt, tsx

## 実装方針

1. `npm init -y` → package.json を編集
2. 必要な型・ランタイム依存を `npm install`
3. tsconfig は `module: ESNext`, `moduleResolution: Bundler`, `strict: true`
4. scripts:
   - `dev`: 並行実行 (`tsx watch src/server/index.ts` と `vite`)
   - `build`: `vite build && tsc -p tsconfig.server.json`
   - `start`: `node dist/server/index.js`
   - `migrate`: `tsx scripts/migrate.ts`
   - `seed`: `tsx scripts/seed.ts`
   - `lint`: `biome check .`
   - `test`: `vitest run`
