# シフト編成画面 A02

## 背景

店長が週ビューでシフトを編成・公開する画面。最も操作頻度の高い管理機能。

## 追加するファイル

- `src/client/routes/AdminShifts.tsx`
- `src/client/components/ShiftGrid.tsx` (縦軸=従業員, 横軸=日付 7 日)
- `src/client/components/ShiftCellEditor.tsx` (start/end 時刻入力, draft/published 表示)
- 任意: dnd-kit を導入してドラッグでセル間コピー

## 実装方針

- 上部: 週切替 (前週/翌週/今週) + 店舗タブ + 「下書き保存」「公開する」ボタン
- グリッド: 従業員行 × 7 日列。空セルクリックでエディタ open、既存セルクリックで編集/削除
- 重複検出は API 409 をハンドルし朱色でエラー表示
- 人員不足日は列ヘッダに警告アイコン (`/api/shifts/conflicts` で取得)
- ドラフトと公開済みは視覚的に区別 (ドラフト=点線枠, 公開=実線 + 金茶)
- 「公開する」ボタンで `/api/shifts/publish` 呼び出し、確認ダイアログ「{N} 件のシフトを公開します。よろしいですか？」
