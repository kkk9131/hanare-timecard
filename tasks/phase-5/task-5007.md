## A03: 修正申請審査

- 未処理 / 処理済みタブ
- 各申請カードに: 申請者, 対象日時, 申請内容 (旧→新), 理由
- 「承認」「却下」ボタン。却下時は理由必須モーダル
- 承認/却下後一覧が更新される

## A04: 従業員マスタ

- 一覧表 (氏名 / カナ / 役割 / 所属店舗 / 時給 / 状態)
- 「新規追加」「編集」「PIN リセット」「退職」ボタン
- フォーム: 氏名/カナ/role/login_id/password (manager+ のみ)/PIN/時給/入社日/所属店舗複数選択
- 退職処理は確認ダイアログ「{name}さんを退職扱いにします。よろしいですか？」

## A05: 店舗マスタ

- 一覧 + 編集モーダル (名称, 表示名, 営業時間, 定休日)
