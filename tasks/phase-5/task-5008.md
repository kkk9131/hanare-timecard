- 期間カスタム指定 (from/to date picker)
- 店舗選択 (雀庵 / 雀庵はなれ / 全店舗)
- xlsx / CSV の 2 種類ボタン
- ダウンロード時はブラウザ標準の保存ダイアログに任せる
- ダウンロード完了後「{filename} を保存しました」トースト

- ページング (limit 50)
- 削除ボタンは存在しない (read only)
