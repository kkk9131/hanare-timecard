# task-5003 done_when

- `/` で従業員一覧タイルが表示される
- タイルをタップ → `/punch/pin` で PIN パッドが表示
- 正しい PIN で `/punch/board` に遷移
- 状態 off ではボタン「出勤する」のみ enabled、他は disabled
- 状態 working では「退勤する」「休憩に入る」のみ enabled
- 「退勤する」を押すと「本当に退勤しますか？(HH:MM)」確認が出る
- 打刻成功後 K04 で「○○さん、お疲れさまです。退勤を記録しました（HH:MM）」が表示される
- K04 から 5 秒後に `/` に戻る (logout 済み)
- 誤 PIN 5 回でロックメッセージが表示されカウントダウンが進む
- iPad 縦 (768×1024) で全画面が破綻しない
- 打刻ボタンが 56×56 以上で文字サイズが 20px 以上
- 1 タップから完了まで 3 ステップ (名前 → PIN → 打刻) で完結する
