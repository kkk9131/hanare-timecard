# 従業員ダッシュボード E01-E05

## 背景

従業員が自分の勤怠・履歴・修正申請・シフトを確認する画面群を実装する。

## 追加するファイル

- `src/client/routes/MeDashboard.tsx` (E01: 当月集計、状態カード)
- `src/client/routes/MeHistory.tsx` (E02: 打刻履歴 月切替, 過去 3 ヶ月)
- `src/client/routes/MeCorrections.tsx` (E03: 申請一覧 + 新規申請フォーム)
- `src/client/routes/MeShifts.tsx` (E04: 公開シフト カレンダ表示)
- `src/client/routes/MeShiftRequests.tsx` (E05: 希望提出フォーム + 一覧)
- `src/client/api/corrections.ts`
- `src/client/api/shifts.ts`

## 実装方針

- AppShell を使った左サイドバーナビ (打刻ボード / 履歴 / 修正申請 / シフト / 希望)
- 集計値は WashiCard 内に大きく表示 (実働 ◯時間◯分 / 残業 ◯時間◯分 / 休憩 ◯時間◯分 / 深夜 ◯時間◯分)
- 履歴は表形式 (日付 | 出勤 | 退勤 | 休憩 | 実働)
- 修正申請: 対象日 + 修正後時刻 + 種別 + 理由
- 状態は StatePill で表示 (申請中/承認/却下)
