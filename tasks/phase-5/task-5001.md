# React 骨組とルーティング

## 背景

`src/client/` に React + Vite + react-router-dom + TanStack Query の骨組を作る。各画面はプレースホルダで OK。

## 追加するファイル

- `src/client/main.tsx` (QueryClientProvider, BrowserRouter)
- `src/client/App.tsx` (Routes 定義)
- `src/client/api/client.ts` (fetch ラッパ + zod パース + credentials:'include')
- `src/client/routes/` 以下に各画面のプレースホルダコンポーネント (K01-K04, E01-E05, A00-A07)
- `src/client/components/AuthGuard.tsx` (role 制限)

## 実装方針

- パス定義は ui-design.md の画面一覧に一致
- AuthGuard は `/api/auth/me` を fetch して未認証なら `/` または `/admin/login` にリダイレクト
- TanStack Query デフォルト: staleTime 30s
- 各画面コンポーネントは中央に画面名を表示するだけで OK (この時点)
