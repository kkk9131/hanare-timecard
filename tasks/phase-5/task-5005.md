# 管理者ログインとダッシュボード

## 追加するファイル

- `src/client/routes/AdminLogin.tsx` (A00: ID + PW フォーム)
- `src/client/routes/AdminDashboard.tsx` (A01: 店舗切替, 当日勤務状況, 未処理申請件数, 今週シフトサマリ)
- `src/client/components/StoreSwitcher.tsx` (雀庵/雀庵はなれ/全店舗)
- 管理画面用 AppShell の sidebar 構成 (ダッシュボード/シフト/修正申請/従業員/店舗/エクスポート/監査)

## 実装方針

- A00: ID/PW フォーム + 「管理者ログイン」ボタン。失敗時丁寧語エラー
- A01:
  - 店舗切替タブ (StoreSwitcher、雀庵/雀庵はなれ/全店舗)
  - 「現在勤務中の従業員」リスト (名前 + 出勤時刻)
  - 「未処理の修正申請」件数 (リンク → A03)
  - 「今週公開済みシフト」サマリ
  - 「今すぐエクスポート」へのショートカット
- 兼務従業員のフィルタは store_id を含めて API へ渡す
